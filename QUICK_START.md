# Quick Start Guide - SPA VIP Management

## Installation (First Time)

1. **Download and Install**
   - Download `SpaApp-Setup-1.0.0.exe` from the release page
   - Run the installer
   - Follow the installation wizard

2. **First Launch**
   - Click "SPA VIP Management" from Start Menu or Desktop
   - Wait for database to initialize (first time only)
   - App is ready to use!

## Daily Usage

### 1. Dashboard
- View today's revenue
- Check upcoming bookings
- Monitor key statistics

### 2. Booking Management
- Click "Dat Lich" menu
- Add new appointments
- Assign staff members
- Track booking status

### 3. Payment Processing
- Click "Thanh Toan" menu
- Select customer
- Choose service/product
- Select payment method
- Complete transaction

### 4. Inventory
- Click "Kho Hang" menu
- Add/edit products
- Track stock levels
- Monitor low stock alerts

### 5. Customer Management
- Click "Khach Hang" menu
- Add new customers
- View customer history
- Manage loyalty points

### 6. Staff Management
- Click "Nhan Vien" menu
- Add staff members
- Set commission rates
- View staff performance

### 7. Reports
- Click "Bao Cao" menu
- Select date range
- Export to Excel or PDF
- Analyze business metrics

## Common Tasks

### Export Data to Excel
1. Go to desired module (Bookings, Customers, etc.)
2. Click "Xuat Excel" button
3. Choose save location
4. File saved as `.xlsx`

### Backup Database
1. Go to Settings
2. Click "Backup Now"
3. Backup saved to %APPDATA%\SpaApp\backups\

### Switch Theme
- Click theme icon (circle/sun icon) in top-right corner
- Switch between Light and Dark mode

### Search
- Use search box in each module
- Filter by name, phone, date, etc.

## Tips & Tricks

- **Keyboard Shortcuts**:
  - Ctrl+S: Save
  - Ctrl+Q: Quit
  - Ctrl+N: New record

- **Responsive Design**: App works on laptops and tablets

- **100% Offline**: No internet needed, all data stored locally

## Troubleshooting

### App won't start?
1. Restart computer
2. Reinstall application
3. Check Windows Administrator rights

### Lost data?
- Check backup folder: `%APPDATA%\SpaApp\backups\`
- Restore from latest backup

### Performance issues?
1. Restart application
2. Clear temporary files
3. Ensure sufficient disk space

## System Requirements

- Windows 7 or later
- 4GB RAM (minimum)
- 500MB disk space
- Internet (optional, offline works)
